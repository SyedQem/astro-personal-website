## Getting Started

Clone the repo and install dependencies:

```bash
npm install
```

Run the dev server:

```bash
npm run dev
```

Open `http://localhost:3000` in your browser.

---

## Scripts

```bash
npm run dev       # start dev server
npm run build     # production build
npm run start     # start production server
npm run lint      # lint
```

---

## Deployment

Deployed on **Vercel**. Push to your main branch and it auto-deploys (assuming Vercel is connected).
