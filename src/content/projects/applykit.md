## Overview
Applykit is an ongoing project focused on making the job search process more organized and more intentional. The goal is to build a system that helps track applications across different stages while also supporting resume optimization so each application can be tailored more effectively.

## What I’m building
- A centralized workflow for tracking job applications, statuses, deadlines, and follow-ups
- Resume optimization tools that help align application materials with a given role
- A clean interface that keeps the process focused, readable, and easy to update over time

## Design notes
- **Workflow-first:** designed around the actual steps involved in applying, following up, and refining materials.
- **Clarity over clutter:** keeps important application details visible without making the experience overwhelming.
- **Built to evolve:** since it is an ongoing project, the structure is meant to support iteration as new features and insights are added.

## Current direction
- Continuing to expand the tracking flow and resume-tailoring experience based on how the project evolves in real use
